# FastAPI 
It demonstrates how to build a small **E-commerce API** using FastAPI with:

* GET endpoints
* Query parameters
* Path parameters
* POST requests
* Pydantic validation
* Business logic
* Order processing
* Swagger testing

* Product catalog API
* Product filtering using query parameters
* Product search functionality
* Store summary dashboard
* Customer feedback system
* Bulk order processing
* Order status tracking
* Data validation using **Pydantic**


# 🚀 How to Run the Project


pip install fastapi uvicorn
 Run the FastAPI server

cmd prompt
uvicorn main:app --reload
  Open Swagger UI
http://127.0.0.1:8000/docs
